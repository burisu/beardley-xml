# Beardley::XML

XML jars for Beardley:

  * Castor
  * DOM4j
  * Jaxen
  * SAAJ
  * Xalan
  * Xerces
  * XML APIS

## Installation

Add this line to your application's Gemfile:

    gem 'beardley-xml'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install beardley-xml
